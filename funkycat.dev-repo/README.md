# funkycat.dev

My personal portfolio site. Dark, aviation-themed, no nonsense.

Live at **[funkycat.dev](https://funkycat.dev)**.

## What's here

A single-page static site — plain HTML/CSS/JS, no framework, no build step. It shows what I'm working on:

- **Altimeter Tech** — a Roblox-focused tech brand + Discord bot
- **useformat** — a private, browser-based file converter ([useformat.app](https://useformat.app))
- **Luma** — a media downloader (in development)

## Stack

Nothing fancy on purpose:

- Hand-written HTML + CSS (custom properties, scroll-reveal animations)
- A bit of vanilla JS for the fade-in-on-scroll effect
- Fonts: Syne, Bricolage Grotesque, DM Mono
- Hosted on a Hostinger VPS behind Cloudflare

## Running locally

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
